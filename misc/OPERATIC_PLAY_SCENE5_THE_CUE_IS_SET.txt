============================================================
ACT II — THE LSOF PROBLEM

[Skippy begins to wire up "played" tracking via lsof events.]

CAPTAIN:
♩ Wait. We can't rely on that.
  I load tracks and swap them constantly.
  The lsof fires every time.
  Half those tracks never played a bar. ♩

SKIPPY:
♩ You're right. The clock is the answer.
  If a track stays in the deck
  longer than the other deck's song
  plus two minutes — it played.
  If not — it was just a candidate. ♩

CAPTAIN:
♩ That's exactly right. ♩

[_check_played() is born.
 load_times{} records when each file first appeared.
 deck_track{} holds the current Track per deck.
 When a file leaves: time_open vs other_deck.duration + 120s.
 Solo fallback: 90 seconds.
 False positives: greatly reduced.]

SKIPPY:
♩ Logged. Noted. Documented.
  The NML holds CUE_V2 elements.

  TYPE 3 is Load. TYPE 2 is fade.

CAPTAIN:
♩ Build it. ♩

[stage10_autocue.py is written.
 Two modes: fast (NML math only) and audio (librosa).
 Cue 1 Load at the first sound.
 Cue 2 First Beat at the grid anchor.
 Cue 3 Vocal: harmonic separation, 300–3500 Hz, first sustained onset.
 Cue 4 Drop Loop: rolling onset energy, TYPE=5, one bar.
 Cue 8 Outro: 16 beats from end, TYPE=2, fade.
 Slots 5, 6, 7: reserved. 7 will be last vocals, when the time comes.]
